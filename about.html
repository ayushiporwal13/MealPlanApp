Models

Create at least one Guest object with required fields and at least one MovieWatchlist object with the required fields.

Guests

The Guests data model stores guest account information.

  • _id: A unique identifier for each guest.
  • username: The username of the guest.
  • password: The password for the guest account.
  • movie_watchlist: An array containing the _ids of items in MovieWatchlist.

MovieWatchlist

The MovieWatchlist data model stores information about each movie in the watchlist.

  • _id: A unique identifier for each movie entry.
  • status: The status of the movie ('Watched', 'Pending').
  • movie: An object containing details about the movie, which includes:
    • title: The title of the movie.
    • genre: The genre of the movie.
    • image: The image or poster of the movie.
    • movie_id: The unique TMDB API ID for the movie.

API Endpoints

Design and implement the following endpoints in your Node.js Application. These endpoints must follow REST principles, including proper use of HTTP methods, resource naming, status codes and appropriate URL structures for path and query parameters.

Users

  • Register a new user account
  • Requires a unique username and password
  • Optionally allows specifying dietary preferences
  • Login a user: Validates that the username and password match
  • Retrieve user information by id: Returns user info (excluding password) and includes all associated meal plans for that user
  • Update user preferences by id: Allows user to update only their dietary preferences

Meal Plans

  • Add a new meal plan and/or add meal to existing plan for a user: Allows guest to create a new watchlist. Allows guest to add a movie to watched using search results from the TMDB API.
  • Update status of watchlist movie: Update the status of a specific movie entry in the watchlist by id.
  • Delete watchlist movie: Delete a movie entry from the watchlist by id.

Movies

  • Search for movies by keyword: Search the TMDB API by keyword.
  • Retrieve popular movies: Get current popular movies from TMDB API.
  • Retrieve movie recommendations based on guest watchlist: Get movie recommendations from TMDB API based on a movie id from the guest's movie watchlist.

Interaction with TMDB API

The application will interact with the TMDB API to search for movies, get popular movies and get recommendations based on a guest's movie watchlist.

Guests can add movies returned from the API to their watchlist.
